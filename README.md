# 🚨 응급처치 음성 챗봇

음성으로 응급상황을 말하면 GPT-4.1-mini가 대화형으로 응급처치 방법을 안내해주는 챗봇입니다.

---

## 📌 주요 기능

- 🎙️ **음성 입력 (STT)** — Google Speech Recognition으로 한국어 음성을 텍스트로 변환
- 🤖 **대화형 응급처치 안내** — GPT-4.1-mini가 상황을 파악하며 단계적으로 처치 방법 안내
- 🔊 **음성 출력 (TTS)** — gTTS로 응답을 음성으로 재생
- 💬 **멀티턴 대화** — 이전 대화 내용을 기억하며 맥락 유지

---

## 🛠️ 기술 스택

| 역할 | 라이브러리 |
|---|---|
| STT | `SpeechRecognition` (Google STT) |
| 응답 생성 | `openai` GPT-4.1-mini |
| TTS | `gTTS` |
| 환경변수 관리 | `python-dotenv` |
| 실행 환경 | Jupyter Notebook (로컬) |

---

---

## 🚀 실행 방법

1. `말씀하세요` 출력되면 마이크에 응급상황 말하기
2. `고마워` 라고 말하면 챗봇 종료

---

## 💬 대화 예시

```
[챗봇] 말씀하세요
[사용자] 팔이 부러졌어요

[챗봇] 팔을 움직이지 말고 부목으로 고정해주세요.
       출혈이 있으면 깨끗한 천으로 눌러주세요.
       혹시 팔에 출혈이 있나요?

[사용자] 네 피가 나요

[챗봇] 깨끗한 천으로 출혈 부위를 강하게 눌러주세요.
       5~10분간 유지해주세요.
       출혈량이 많은가요, 적은가요?
```
